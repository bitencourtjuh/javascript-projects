  <h2 style="color:#333;"> --- Sobre o Repositório</h2>
  <p>
    Este repositório reúne uma coleção de projetos desenvolvidos em 
    <strong>JavaScript</strong>, com o objetivo de praticar e evoluir habilidades em desenvolvimento web.
  </p>

  <p>
    Os projetos foram criados para aplicar conceitos fundamentais como

  </p>

  <p>
    Cada projeto contribui para o desenvolvimento prático e construção de um 
    <strong>portfólio sólido</strong> na área de tecnologia.

  <hr>

  <h2 style="color:#333;"> --- O que é JavaScript?</h2>
  <p>
    O <strong>JavaScript</strong> é uma linguagem de programação essencial para o desenvolvimento web moderno, 
    responsável por tornar páginas <strong>dinâmicas e interativas</strong>.
  </p>

  <p>
    Com JavaScript, é possível:
  </p>

  <ul>
    <li> Criar interações em tempo real (cliques, animações, formulários)</li>
    <li> Manipular elementos da página (DOM)</li>
    <li> Consumir APIs e trabalhar com dados dinâmicos</li>
    <li> Desenvolver aplicações completas (frontend e backend)</li>
  </ul>

  <p>
    Atualmente, o JavaScript está presente em praticamente todos os sites e é uma das linguagens 
    mais utilizadas no mundo da programação.
  </p>
